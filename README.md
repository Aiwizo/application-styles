# Aiwizo Styles

Stylesheets and resources for Aiwizo application design.

## Installation

```
npm install @aiwizo/application-styles
```


## Basic Usage

### React

```jsx
import "@aiwizo/application-styles";

const SomeComponent = () => (
  <div
    style={{
      backgroundColor: "var(--aiwizo-application-blue)",
    }}
  />
);
```
